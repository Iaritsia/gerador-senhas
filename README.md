# gerador-senhas
Ferramenta em Python para gerar senhas fortes com opções de complexidade.
Este projeto está **em desenvolvimento**. Em breve publicarei a versão inicial com geração de senhas fortes, opções de tamanho e complexidade, e salvamento em arquivo.

## Objetivo
Construir uma ferramenta simples e confiável para criação de senhas profissionais.

- Implementação: em andamento
- Publicação: em breve

## Tecnologias previstas
- Python (módulos nativos: `random`, `secrets`, `string`)
